# Admin User Manual

## How to login
1. Open `/admin/login`.
2. Enter the admin email and password.
3. Use Logout from the top bar when finished.

## Dashboard overview
The dashboard shows counts for projects, services, gallery images, and contact messages. It also provides quick links to common CMS sections.

## Company information
Open **Settings > General Settings** to update company name, tagline, email, phone, address, copyright text, and developer credit.

## Logo, favicon, and profile PDF
Use **Settings > General Settings**. Upload optimized logo and favicon images. Upload the company profile as a PDF. Save the form and review the header/footer on the frontend.

## Theme colors and fonts
Open **Settings > Theme Settings** to update colors, font names, size values, spacing, button radius, and custom CSS. Avoid scripts or HTML in custom CSS.

## Social links
Open **Appearance > Social Links** to add, edit, deactivate, or delete social links. Use full `https://` URLs.

## Menu items
Open **Appearance > Menu Items**. Use relative paths like `/about` or safe `https://` URLs. Set target to same tab unless the link goes to another site.

## Footer links
Open **Appearance > Footer Links**. Add important site links and set their order.

## Hero sliders
Open **Homepage > Hero Sliders**. Add heading, text, image, optional button text/link, sort order, and status.

## Feature highlights
Open **Homepage > Feature Highlights**. Add short feature cards with title, icon class or image, text, and order.

## Homepage sections
Open **Homepage > Homepage Sections** to edit About, Project Highlights, Services, and Gallery CTA content.

## Projects
Open **Content > Projects**. Add title, category, slug, featured image, progress, descriptions, featured status, active status, sort order, SEO title, and SEO description.

## Services
Open **Content > Services**. Add title, slug, image/icon, descriptions, active status, order, and SEO fields.

## Gallery images
Open **Content > Gallery Images**. Upload images, add title/category, set status, and order.

## Contact messages
Open **Content > Contact Messages** to view, filter, mark read/unread, or delete messages.

## SEO title/description
Projects and services each have SEO title and SEO description fields. Keep titles concise and descriptions readable.

## Recommended image sizes
- Logo: transparent PNG/WebP, about 240x80 px.
- Favicon: square 32x32 or 64x64 px.
- Hero slider: 1920x900 px.
- Project/service cards: 1200x800 px.
- Gallery images: 1200x800 px or larger with optimized file size.
- Homepage section images: 1000x800 px.

## Common mistakes to avoid
- Do not upload very large uncompressed images.
- Do not paste script tags into custom CSS.
- Do not leave projects/services inactive if they should appear publicly.
- Do not use unsafe links such as `javascript:`.
- Always check mobile view after publishing content.
